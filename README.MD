1、代码重构，使用MVP架构<br/>
2、删除：移除表情资源在线下载<br/>
3、删除：首页分类样式<br/>
4、新增：可以添加多个帐号<br/>
5、新增：点击帖子列表板块名称可快速跳转到相应板块<br/>
6、新增：发表帖子支持添加投票<br/>
7、优化：帖子详情页面，浏览更流畅<br/>
8、优化：其它细节优化<br/>
9、修复：大量bug<br/>
注意：<br/>
1、从该版本开始不再支持安卓版本6.0（不包括）以下<br/>
2、升级该版本会清除本地草稿箱数据，请做好备份<br/>

4、优化：评分界面<br/>
